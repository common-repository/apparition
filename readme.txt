== Description ==

Apparition! shows you specific category of links at the admin bar! Just check the link categories at admin page.

여러분이 지정한 링크를 어드민바에 보여줄 수 있어요.

Official Page : <a href="http://www.sujinc.com/lab/apparition/">Apparition!</a>

== Installation ==

1. Download the plugin zip package and extract it (플러그인을 다운로드하여 압축을 해제해주세요)

1. Put the folder named "apparition" under /wp-content/plugins/ directory (플러그인 디렉토리에 넣어주세요)

1. Go to the plugins page in your Wordpress admin panel and click "Activate" (플러그인 관리 페이지에서 활성화 시키세요)

1. Go to Settings -> Apparition!. (세팅의 Apparition!으로 이동하세요!)

1. Just click the categories that you want to set. (그냥 카테고리만 클릭하세요!)

== Frequently Asked Questions ==

= How can I set target attribute? (타겟 - 새 창에서 열릴지 - 은 어떻게 설정하나요?) =

You can edit link at Admin > Links. There is Target metabox! (어드민의 링크에서 타겟이라는 메타박스가 있어요~)
